## Technical Specs (Add to End of Prompts When Needed)

- Email hero: ~1200x600 px, wide landscape, generous negative space at top for headline overlay
- Section graphic / checklist: Portrait or square, clean for mobile
- Product cards: Square or 4:3
- Always request high resolution / sharp focus
- For A/B testing: Generate 2–3 variations with slight differences (lighting, angle, props)

---

## Workflow Reminder
1. Start with base prefix + specific scene from campaign brief Design Notes.
2. Generate 2–3 options.
3. Select best, request light edits if needed (e.g., "make the split cleaner" or "add more vibrant green on right side").
4. Optimize (compress, correct dimensions, excellent alt text).
5. Version in repo: `campaign-001-hero-v1.jpg`, etc.
6. Upload to Klaviyo.

**Tip:** For maximum brand consistency, always reference real Shopify product photos alongside Imagine-generated supporting visuals (heroes, checklists, app composites, seasonal mood).
