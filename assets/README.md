# Assets

The brand's binary files and the registry of everything we create. This folder is meant to **grow** as Claude Design generates work.

## Folders
- `logo/` — official Yard Mastery logo files + links (see `logo/README.md`).
- `swatches/` — color palette swatch images (`ym-color-palette.png`).
- `references/` — example layouts, screenshots, inspiration, and saved email examples for reference.
- `generated/` — **new assets Claude Design creates go here** (images, graphics, lockups).

## Naming convention (use for every new asset)
```
YYYY-MM-DD_channel_campaign-or-subject_variant.ext
```
Examples:
- `2026-06-19_ig_flagship-summer-feature_4x5.png`
- `2026-06-19_email_memorialday-hero_1200x600.png`
- `2026-07-01_yt_doubledark-thumbnail_1280x720.png`

Rules: lowercase, hyphen-separated words, include the channel (`email`, `ig`, `fb`, `yt`, `x`, `story`, `print`) and the aspect ratio or pixel size. Keep editable/source files alongside exports when possible.

## When you add an asset
1. Save the file to `generated/` (or `logo/`, `swatches/`, `references/` if that's its home).
2. Add a row to `ASSET-LOG.md`.
3. If it introduces a reusable pattern or prompt, also update the relevant `guidelines/` or `prompts/` file.
